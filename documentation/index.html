Folder Structure

Before starting any project, it is very important to set a folder structure. It helps us organize our code neatly and makes it easier for the future developers to modify anything in the theme.

Installation Guide:
  • To make sure that you can setup your project smoothly, we have provided an Installation guide for you to follow.
  • This Installation guide contains everything from installing node till running your project. Do not miss any steps while creating a new project or else you might have to start over again.
  • Make sure that you go through the explanation of folder structure used in our theme, it will become easier for you to make any modifications.
  • If you want , you could follow our folder structure in your project , that way if you encounter any difficulties in your project, you can refer our theme.
